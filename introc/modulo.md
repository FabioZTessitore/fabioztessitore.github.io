# L'operatore modulo

Torna all'[indice](../toc.md)

---

Oltre alle classiche quattro operazioni aritmetiche, tutti i linguaggi di programmazione mettono
a disposizione una quinta operazione: il _resto della divisione_, detto anche _modulo_.

```c
7 % 4   // risultato 3

10 % 2  // risultato 0
```
